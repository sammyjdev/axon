# AXON Landing Page

Static single-page site for the AXON project. No build step or framework required — just plain HTML and CSS.

**Preview locally:** run `python3 -m http.server` from this `site/` directory, then open `http://localhost:8000` in your browser.

**Deployment** (e.g. GitHub Pages) is a manual step performed by the project maintainer; it is not automated by this repository.
